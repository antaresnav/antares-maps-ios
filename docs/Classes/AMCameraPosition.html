AMCameraPosition

public final class AMCameraPosition: Hashable, CustomStringConvertible

An immutable class that aggregates all camera position parameters such as location, zoom level and bearing. Use AMCameraPosition.Builder to construct a AMCameraPosition instance, which you can then use in conjunction with AMCameraUpdateFactory.
